# Code Of Conduct

Arena is a public project for people building, testing, and watching autonomous game bots. Keep collaboration technical, specific, and respectful.

## Expected Behavior

- Be welcoming to new contributors.
- Assume good intent, but be precise when reviewing code.
- Keep criticism focused on the work, not the person.
- Respect maintainer decisions about scope, roadmap, security, and moderation.
- Do not publish private credentials, private infrastructure details, or undisclosed security findings.

## Unacceptable Behavior

- Harassment, threats, slurs, or sustained personal attacks
- Sexualized comments or imagery in project spaces
- Trolling, spam, or repeated off-topic disruption
- Publicly disclosing vulnerabilities before maintainers have had a reasonable chance to respond
- Posting another person's private information without permission

## Reporting

Report conduct concerns to the maintainers through GitHub. For security-sensitive conduct or vulnerability issues, use the private vulnerability reporting path described in [SECURITY.md](SECURITY.md).

Maintainers may remove content, lock threads, block users, or take other moderation action when needed to protect the project.
