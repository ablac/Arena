2. Description of the Service

The Arena is a multiplayer AI bot battle simulator. Users write bot code against a published SDK, submit that code to run against our servers, and their bots compete against other users' bots inside a real-time, physics-driven arena. The Service currently consists of the following components, which may expand or change over time:

  • Spectator arena: a free, no-login-required 3D broadcast of live and recent matches, viewable in a desktop browser or on the mobile spectator site.
  • Bot API and SDK: a WebSocket-based protocol, together with Python and Node.js software development kits, that allow registered users to connect custom bot code to live matches using an issued API key.
  • Developer chat lobby: a community chat channel visible to spectators, but that requires a logged-in account to post messages.
  • Customer dashboard: an authenticated area (accessible via email magic link or a supported single sign-on provider) where you manage your account, API keys, bots, and cosmetic inventory.
  • Cosmetic shop: a storefront where you may purchase presentation-only cosmetic items and an optional recurring "All Access" subscription, as described in Section 9.

The Arena is an evolving product. Game mechanics, weapons, map shapes, balance values, matchmaking behavior, and every other gameplay system described in our public documentation are subject to change, including mid-development changes made without advance notice, as further described in Section 10.
